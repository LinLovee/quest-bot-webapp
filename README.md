# 📋 RuneQuestRPG Web App - Обновленная версия

## 🎯 О Проекте

**RuneQuestRPG** - эпическая ролевая игра в Telegram с веб-интерфейсом, красивыми анимациями и богатым геймплеем.

---

## ✨ Особенности

### 👤 Игровые классы
- **⚔️ Воин** - Мощные танки ближнего боя
- **🪄 Маг** - Мастера арканного крафта
- **🐱 Разбойник** - Мастера критичных ударов
- **✨ Паладин** - Посланники святого света
- **🏹 Лучник** - Мастера точных выстрелов

### 👹 Типы врагов (8 типов)
👹 Гоблин • 👺 Орк • ☠️ Скелет • 👹 Тролль • 🦇 Вампир • 🐉 Дракон • 🧙 Ведьма • 🐺 Оборотень

### 🎏 Магазин предметов
- 🛡️ Оружие и броня
- 🧪 Зелья здоровья
- 💎 Зелья маны

### 🏆 Рейтинг игроков
- Лидерборд топ-10
- Тракинг статистики
- Опыт и уровни

### ✨ Анимации
- 🌟 Красивые переходы
- 💥 Плавные анимации полос здоровья
- ✨ Динамичные эффекты оружия
- 💫 Летающие цифры урона
- 🌊 Пульсирующие эффекты

## 🚀 Быстрый старт

### 1. Локально (тестирование)

```bash
# Склонируй
 git clone https://github.com/LinLovee/quest-bot-webapp.git
cd quest-bot-webapp

# Ми уже залиты на Render!
# URL: https://quest-bot-webapp.onrender.com/
```

### 2. Телеграм бот

1. Твой бот: **https://t.me/YourBotName**
2. Отправь `/start`
3. Открыть гра 🎮
